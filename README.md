# Our Garden

A little living page. One seed at a time — water it once a day, watch it grow from seed to full bloom, and leave a note if you want. Miss a day and it still gets watered, with a message waiting when you're back.

Once something's fully grown, it moves to the garden and a new seed unlocks.

Built for just the two of us.

## Setup

Just open `index.html` — no build step, no dependencies. Works as a static GitHub Pages site.
